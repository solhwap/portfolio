스마트 화분

앱 인벤터로 제작 된 화분관리 앱(프로토타입)입니다. 오염 물질을 간이정수기를 이용하여 정수처리, 정수처리 된 물은 비접촉 수위센서로 측정하여 진행 상황을 알 수 있습니다. 원격으로 화분에 물을 주고, 식물 정보를 제공합니다. 계절학기 수강 중 빠르게 아이디어를 구상하고 앱의 각 화면 UI, 검색 페이지의 기능만을 간단히 구현하였습니다.

참여인원 : 6명
기여도 : ★★★★☆
제작기간 : 2020년 12월 28일 ~ 2020년 1월 18일

동작인식 알람앱

안드로이드 스튜디오, OpenCV(Haar Cascades검출기 파일 사용)를 이용하여 제작하였습니다. 검출기가 사용자의 눈동작을 감지하여 눈을 감으면 알람음이 계속 재생되고, 일정 시간 동안 눈을 계속 뜨고 있으면 알람음이 더 이상 울리지 않는 방식의 알람 앱입니다. 사용자가 알람을 설정한 시각이 되면 알람이 크게 울리고, 알람 목록을 길게 누르면 사용자의 모션을 인식하는 화면으로 넘어갑니다. KeyEvent를 설정하여 볼륨음을 기기 최대 값으로 설정하고 알람이 중지되기 전까지 사용자가 알람음을 임의로 줄일수 없게 구현하였습니다. 알람 설정 화면은 안드로이드 스튜디오의 TimePickerDialog와 DatePickerDialog 대화상자 기능 등 활용하여 구성하였습니다.
